## Key Takeaways

- **Amazon RDS** is a managed relational database service that supports multiple engines (MySQL, PostgreSQL, etc.). AWS handles patching, backups, and high availability.

- **Amazon Aurora** is a high-performance relational database built by AWS. It offers better speed, automatic storage scaling, and higher availability than standard RDS while remaining compatible with MySQL and PostgreSQL.

- **Amazon DynamoDB** is a fully managed NoSQL database designed for extremely high throughput and low latency at scale. It automatically scales and is ideal for applications needing fast, flexible data access.

- **Amazon Redshift** is a managed data warehouse used for large-scale analytics and business intelligence workloads.

- **Amazon Athena** allows users to run SQL queries directly on data stored in S3 without loading it into a database first.

- **Amazon ElastiCache** provides in-memory caching (Redis or Memcached) to improve application performance by reducing load on primary databases.

- **Amazon Kinesis** is used for real-time data streaming and processing large volumes of data as it arrives.

- **Amazon QuickSight** is a business intelligence service used to create dashboards and visualizations from company data.
