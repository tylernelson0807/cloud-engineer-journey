- Began the **Other Compute Services** section, which introduces several important compute options beyond traditional EC2:
  - **AWS Lambda**: Serverless compute service that allows running code without provisioning or managing servers.
  - **Amazon Lightsail**: Simplified virtual private server (VPS) offering designed for ease of use and predictable pricing.
  - **AWS Batch**: Managed service for running large-scale batch computing workloads efficiently.
  - **Amazon ECS (Elastic Container Service)**: Fully managed container orchestration service for deploying, managing, and scaling containerized applications.

- These services expand the range of compute options available and are commonly tested on the Cloud Practitioner exam, especially in scenario-based questions involving serverless, containers, and simplified deployments.
